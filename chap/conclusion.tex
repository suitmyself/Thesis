\specialchap{结论}
利用PMO观测\COU 和 \COL 所得数据，并结合其他波段的已有数据，本文对红外bubble N37 进行了一项多波段的观测研究。上述章节的详细讨论，可以总结为以下几点：
\begin{enumerate}
  \item 虽然\mycite{Churchwell}{2006}将其归为B类，但经过我们的研究，本文认为其拥有更为复杂的形态结构，在N37的北边，另有两个与之成协的bubble N37―1和N37―2。这三个bubble在物理动力学上存在相互作用，相互影响其演化过程。其中N37本身演化程度较高，而N37-1和N37-2演化程度较低。
  \item 本文确认区间为$38\sim43\kms$的速度成分与bubble成协，其系统速度为$41.2\kms$，与已有研究的结论是一致的。PMO的观测数据在此速度区间揭示的是N37东北边(N37-1东边及南边)的结构。而\JCMTCO 的数据虽然其并未完全覆盖N37-1，但其很好的展示了N37的形态结构，和GLIMPSE 8.0um的数据高度一致。
  \item 利用PMO \COL 的数据，本文找到了4个致密核，其平均数密度为$1.5\times10^3\textrm{cm}^3$，平均质量为$1.1\times10^3\Msun$，bubble的总质量为$\sim1.8\times10^4\Msun$。四个core的的平均速度展宽$\sim1.7\kms$，暗示其中可能存在大质量星的形成过程。利用\JCMTCO 的数据，我们找到了6个致密的核，其平均速度展宽为$\sim 2.8\kms$，如此大的速度展宽，表明N37的周围的致密核正在形成大质量星。
  \item 利用GLIMPSE-I Spring '07 Point-source catalog的点源数据，本文证认了8个Class I的原恒星和45个Class II 的年轻星。大部分原恒星在N37―1的边缘，与周围的大量冷尘埃物质成协。在N37的内部，出现Class II 的年轻星聚集的现象。大部分年轻星其要分布在bubble的内部或边缘处，表明这些区域存在恒星形成过程。
  \item N37和一个已知的\HII 区成协，在N37的内部充满了24um辐射和20cm辐射，其等效于一个O9.5的大质量星，其在主导bubble的形态结构。\JCMTCO 的数据表明在N37的内侧，存在挤压现象，其六个cores 的线心速度也表明，N37的壳层在持续膨胀，这有可能是\HII 区的扩张导致。
  \item 考虑到Bubble周围尘埃物质的聚集（特别是N37-1），和周围远高于其他区域的年轻星分布密度，表明由于\HII 区的扩张和星风作用，N37的周围存在触发恒星形成过程。比较N37的动力学年龄和致密核的碎裂时标，其并不符合所谓的collect \& collapse 模型。
\end{enumerate}

\chapter{结论}
利用PMO观测\COU 和 \COL 所得的分子转动谱数据，并结合其他波段的已有数据，本文对红外bubble N37 进行了一项多波段的观测研究。根据上述章节的讨论，本文可以总结为以下几点：
\begin{enumerate}
  \item 虽然\mycite{Churchwell}{2006}将其归为B类，但经过较为详细的分析，本文认为其拥有更为复杂的形态结构，在N37的北边，另有两个与之成协的bubble N37$-$1 和N37$-$2。这三个bubble在动力学上存在相互作用，相互影响其演化过程。其中N37本身演化程度较高，在其内部和周围探测到大量的年轻星聚集的现象，而N37$-$1 和N37$-$2 演化程度稍低，在其周围观测到一定数量的原恒星。其中和N37接壤的N37$-$2内部存在24um辐射，其极有可能是由N37 主导形成的。
  \item 本文确认区间为$38\sim43\kms$的速度成分与bubble成协，其系统速度为$41.2\kms$，这与已有研究的结论是一致的。PMO的观测数据在此速度区间揭示的是N37东北边(N37-1东边及南边)的聚集团块。而\JCMTCO 的数据虽然未完全覆盖N37-1，但其很好的展示了N37的结构，和GLIMPSE 8.0um的数据展现的bubble的轮廓高度一致。
  \item 利用PMO \COL 的数据，本文找到了4个致密核，其平均数密度为$1.5\times10^3\textrm{cm}^3$，平均质量为$1.1\times10^3\Msun$，bubble的总质量为$\sim1.8\times10^4\Msun$。四个core的的平均速度展宽$\sim1.7\kms$，暗示其中可能存在大质量星的形成过程。利用\JCMTCO 的数据，本文找到了6 个致密的核，其很好的拟合了bubble N37的轮廓，6个core的平均速度展宽为$\sim 3.0\kms$，对应速度的积分强度图表明在N37内侧的致密程度大于bubble的外侧，存在明显的挤压现象，其六个cores 的线心速度也表明，N37的壳层在持续膨胀，这可能是\HII 区的扩张的结果。
  \item 结合GLIMPSE-I Spring '07 Point-source catalog的点源数据，本文证认了8个Class I的原恒星和45个Class II 的年轻星。大部分Class I原恒星在N37$-$1 的边缘，与周围的大量冷尘埃物质成协。在N37的内部，探测到Class II 的年轻星聚集的现象。包括ClassI 和 Class II 的大部分年轻星主要分布在N37、N37$-$1、N37$-$2 的内部或边缘处，表明这些区域正形成新一代恒星。
  \item N37和一个已知的\HII 区成协，在N37的内部充满了24um辐射和20cm辐射，其等效于一个O9.5的大质量星激发，主导了整个N37 bubble的形态结构。
  \item 考虑到Bubble周围尘埃物质的聚集（特别是N37-1），和远高于其他区域的年轻星分布密度，说明由于\HII 区的扩张和星风作用，N37、N37$-$1、N37$-$2 三个bubble 的周围存在触发恒星形成过程。但比较N37的动力学时标和致密核的碎裂时标，其并不符合所谓的collect \& collapse 这一理想模型。
\end{enumerate}
